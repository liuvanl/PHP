# 第十章排序算法
> 说明：排序就是对某组数据进行升序或降序的排列方式，排序都是对索引数组
## 10.1冒泡排序
+ 从大到小排序--从小到大排序
> 找到从最大下标的值和前面的值进行比较，如果前面小于后面的值，进行位置交换，以此类推一直找当前最大的值，一次完成。继续找第二大下标的值进行比较，和它前面的值交换位置，以此类推，最终排序完成
+ 案例：把数组从大到小进行排序
```php
  <?php
    // 从大到小排序
    $arr = array(3,4,2,50,30);
    
    for($i=0;$i<=count($arr)-1;$i++){
      for($j=0;$j<count($arr)-1;$j++){
        if($arr[$j] < $arr[$j+1]){
          $temp = $arr[$j+1];
          $arr[$j+1] = $arr[$j];
          $arr[$j] = $temp;
        }
      }
    }    
    var_dump($arr);
  ?>
```
+ 案例：把数组从小到大进行排序
